# 🎯 Number Guessing Game

A simple command-line Python game where the player tries to guess a randomly generated number within a limited number of attempts.

## 📖 Description

The program generates a random number, and the player must guess it correctly. After each guess, the game provides feedback indicating whether the guess is too high or too low until the correct number is found.

This project demonstrates the use of:
- Random number generation
- Loops and conditional statements
- User input handling
- Basic game logic in Python

## 🚀 Features

- Random number generation
- User-friendly gameplay
- Hints for incorrect guesses
- Unlimited or limited attempts (depending on implementation)
- Simple and lightweight console application

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/SwapnilSahu0909/Number-guessing-game.git
```

2. Navigate to the project folder:

```bash
cd Number-guessing-game
```

3. Run the program:

```bash
python main.py
```

> If your main file has a different name, replace `main.py` with the correct filename.
